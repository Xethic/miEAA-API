# Installing miEAA

Dependencies:

* Python >= 3.5
* Requests >= 2.19

#### Python Package Index

```
$ pip install mieaa
```

#### Conda [![Anaconda-Server Badge](https://anaconda.org/conda-forge/skidl/badges/installer/conda.svg)](https://anaconda.org/ccb-sb/mieaa)

```
$ conda install -c ccb-sb mieaa
```
